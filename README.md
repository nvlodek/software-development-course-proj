# Process Feedback Website Reconstruction

A course project at the University of Missouri–St. Louis (UMSL) — Software Engineering.

## Live Site

[https://software-development-course-proj.vercel.app/](https://software-development-course-proj.vercel.app/)

## About

This project is a reconstruction of the Process Feedback public-facing website using React. It was created solely for academic purposes and is not affiliated with or endorsed by Process Feedback, LLC.

Content from [www.processfeedback.org](https://www.processfeedback.org)

## Tech Stack

- React
- React Router
- CSS
- Formspree (contact form)

## Running Locally
```
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.
